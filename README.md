# Kermit-95
Kermit 95 is a terminal emulator for Windows and OS/2 with a long history. For the last several years I've been its maintainer.

For further information, see:
  - [The Kermit 95 Website on GitHub](https://davidrg.github.io/ckwin/)
  - [The Kermit 95 beta home page on kermitproject.org](https://kermitproject.org/ckw10beta.html)
  - [The repository on github](https://github.com/davidrg/ckwin)
  - [Development log](https://www.ext.zx.net.nz/software/notes/kermit95/)

## Control Sequences Documentation
As part of my work on Kermit 95, I've tried to document the control sequences it supports. This information may also be of interest to users of other terminals:
  - [Control sequences supported by Kermit 95, and what they do](https://davidrg.github.io/ckwin/dev/ctlseqs.html) - originally modeled after xterms documentation
  - [A big but not comprehensive list of known control sequences](https://davidrg.github.io/ckwin/dev/all-ctlseqs.html) - complete with references!
  - [A page comparing availability of control sequneces between DEC VT terminals and a few terinal emulators](https://davidrg.github.io/ckwin/dev/term-ctlseqs.html)
